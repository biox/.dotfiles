# Usage

- Install a new NixOS system
- `git clone git@github.com:biox/.dotfiles.git`
- `cp configuration.nix /etc/nixos/configuration.nix`
- `nixos-rebuild switch`
- `cd .dotfiles`
- `stow [vim|reshift|zsh|etc]`
